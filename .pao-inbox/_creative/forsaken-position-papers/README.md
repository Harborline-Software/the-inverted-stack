# Forsaken Position Papers - Voice Reference Library

This directory holds **complete in-universe documents** from the centralizing direction (Stefan Reinhardt; Helvetia Trust SA; the broader federated-trust local-first ecosystem). Each document is sincere, operationally informed, and architecturally serious. Together they establish the **voice register** for any subsequent chapter epigraph drawn from the Forsaken side.

## Why these exist (per the Long Now thread doc)

> *Recurring chapter epigraphs from the fictional world's documents: Joel's engineering papers, Anna's mission logs, council review dissents, the Forsaken's own position papers (they have sincere answers to this question; they're just wrong). Each epigraph lands the theme in a different register before the chapter earns it. The Forsaken's position papers as epigraph sources is particularly useful: their argument is always "we'll preserve your memories better / more safely." The counter is "on your terms, not mine."*

## Series-design constraint - these documents must be plausible

The series' hardest editorial challenge is **making the antagonist's position genuinely persuasive before the counter-argument lands.** A strawman Forsaken makes the architecture's eventual triumph cheap. A sincere Forsaken whose position the reader can almost accept makes the triumph earned.

The constraint:
- **Sincere** - the Forsaken genuinely believes the position
- **Operationally informed** - the position engages with real deployment problems
- **Architecturally serious** - the engineering is competent
- **Subtly compromised** - the centralization is present but not foregrounded; legible to the architecturally sophisticated reader on careful reading; invisible to the casual reader

These exemplars are voice references. Any epigraph drawn from this side of the series should match this register.

## Documents in this directory

1. `01-stefan-op-ed.md` - Stefan Reinhardt op-ed (~500 words; published ~2025; architect's voice)
2. `02-helvetia-public-position.md` - Helvetia Trust SA public position paper (~600 words; corporate-PR voice; Swiss-neutrality branding)
3. `03-regulatory-filing.md` - formal filing with a regulatory body supporting federated-trust as compliance reference architecture (~700 words; institutional voice)

Each document includes a header capturing publication context (where, when, who signed it) so that epigraphs drawn from it can be properly attributed in-universe.
