# Stage 3 — Package Design

Per-chapter (or per-section) design before scaffolding. The "package" is the bundle of decisions that a drafter needs to produce a faithful skeleton.

**Input:** Approved architecture.

**Artifacts:**
- `NN-short-slug/design.md` — target word count, voice register, chapter type (specification / tutorial / council / two-act / narrative), audience tilt, structural skeleton (scenes or sections), beats list, citation plan, code-snippet inventory, diagrams planned.
- `NN-short-slug/voice-plan.yaml` *(optional)* — per-section voice agent assignments (e.g., `voice-lencioni` for procurement scene, `voice-gladwell` for case study).

**Templates:** `templates/chapter-{standard,council,reference,playbook}.md`.

**Exit criterion:** Drafter can sit down and produce a scaffold without further questions.
