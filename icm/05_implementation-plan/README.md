# Stage 5 — Implementation Plan

How the scaffold becomes a finished draft. Agent assignments, pass order, gates, and review checkpoints. Follows the Universal Planning Framework in `.claude/rules/universal-planning.md`.

**Input:** Approved scaffold.

**Artifacts:**
- `NN-short-slug/plan.md` — UPF-shaped plan: Context & Why, Success Criteria, Assumptions & Validation, Phases (each gated PASS/FAIL), Verification.
- `NN-short-slug/agents.md` *(optional)* — which voice agents touch which sections, in what order.

**Exit criterion:** Plan reaches at least **B (Solid)** on the UPF rubric. Stage 0 discovery is captured. Replanning triggers are defined.
