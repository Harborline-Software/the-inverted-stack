# Stage 2 — Architecture

Structural decisions: where does this change live, what does it displace, what arcs does it touch.

**Input:** Discovery brief.

**Artifacts:**
- `NN-short-slug/architecture.md` — placement, part/act, chapter neighbors, dependency graph, impact on existing chapters, arc/character/world-canon impact (consult `_series/`).
- `NN-short-slug/decision.md` *(optional, when a real choice exists)* — ADR-style: options considered, choice, why, reversibility.

**Exit criterion:** Placement locked. Impact on `book-structure.md` (Vol 1) or Vol 2 outline is enumerated. Cross-volume continuity flagged if relevant.
