# Scoping — Volume-Level Planning Artifacts

This directory holds **volume-level scoping plans** that bridge between the volume
SPINEs (canonical structural commitments) and the per-chapter ICM-stage authoring
work. Scoping plans operationalize the SPINE without superseding it.

## What lives here

| Plan | Volume | Status |
|---|---|---|
| [`vol-3-plan.md`](./vol-3-plan.md) | Vol-3 *The Standards Body* (Wanjiru's volume) | Scoping plan 2026-05-25; awaits Vol-2 ratification before `icm/draft` |
| [`vol-2-finalization-checklist.md`](./vol-2-finalization-checklist.md) | Vol-2 *The Filchner Dark* (Anna's volume) | Working checklist 2026-05-25; tracks open finalization items |

## What this directory is NOT

- Not a substitute for volume SPINEs (`vol-N/SPINE.md` files remain canonical)
- Not a substitute for `.pao-inbox/_state-snapshots/` PAO state files (those track
  point-in-time PAO authoritative reads; scoping plans are forward-looking)
- Not a substitute for `icm/` filesystem pipeline (per-chapter scaffolding lives
  there)
- Not draft prose (no prose lives in this directory)

## Convention

Each scoping plan is a forward-looking authoring guide. It is updated as the
volume's status changes. When a volume ratifies, its scoping plan archives to
`.pao-inbox/_state-snapshots/` with a date-prefix.
